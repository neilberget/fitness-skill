# Programming principles

Heuristics for designing a single session that fits the user's profile, recent log, and today's constraints. Read this before designing a workout if it's not already in context.

## Start from the goals, not the menu

The profile's long-term goals and flexible weekly compass dictate weekly emphasis. Common goal → weekly dose mapping:

- **VO2max ↑** → 1–2 hard interval sessions (4×4 norwegian, 30/30s, 5×3 at ~95% HRmax) + 2–4 zone 2 sessions (45–90 min @ 60–70% HRmax) per week.
- **Muscle mass / ALMI ↑** → 2–4 resistance sessions/week hitting each major pattern (squat, hinge, push, pull, carry) with progressive overload, 8–20 working sets per muscle group per week, most sets in the 5–15 rep range.
- **Strength (1RM) ↑** → lower-rep work (3–6 reps) on main lifts, longer rest (3–5 min), heavier loads (80–95% 1RM).
- **Power** → low-rep explosive work (jumps, throws, Olympic lift derivatives, kettlebell swings) before fatiguing work, fully rested.
- **Mobility / balance / stability** → daily small dose (10–20 min) is more effective than a weekly hour. Embed in warm-ups.
- **Body weight ↑ (lean)** → resistance + caloric surplus; cardio dose moderate, not maximal.
- **Longevity / general capability** → mix of all of the above; don't over-index on one.

A single session typically advances 1–2 of these. Don't try to do everything in 45 minutes.

## Use a weekly compass, not guilt

Most users miss days, travel, sleep poorly, or have uneven equipment access. Treat weekly targets as a compass for choosing the next useful session, not a schedule debt to repay.

- Default to a loose plan with broad weekly targets and 2–4 next-good session options.
- If the user asks for a fixed weekly plan, give exact days, but include fallback rules for missed days.
- Never stack missed workouts into one oversized catch-up day.
- If the week is going sideways, preserve the highest-value minimum dose: one strength exposure, one aerobic exposure, and short mobility where useful.
- When time is short, shrink the session before changing the training intent.

## Rotate, don't repeat

Cross-reference the recent log:

- Heavy lower-body yesterday → upper-body or zone 2 today.
- Two interval sessions in the last 4 days → make today zone 2 or strength.
- Hit the same lift heavy 3 sessions in a row → vary the variation (e.g., trap bar → front squat) or deload.
- Nothing aerobic in 5+ days → add at least a zone 2 finisher.
- Several missed days → resume with the next best session from the weekly compass; don't punish the gap.

## Respect recovery signals

From the log's RPE and notes:

- Two consecutive sessions at RPE 9–10 → today should be RPE ≤ 6 (zone 2, mobility, technique).
- Reported poor sleep / high stress → drop intensity, keep volume modest, avoid CNS-heavy lifts.
- New pain → work around it, do not "push through." Suggest the user update the profile if it persists.
- Chest pain, fainting, severe or unusual shortness of breath, acute injury, neurological symptoms, uncontrolled dizziness, or worsening pain → do not prescribe a workout. Recommend stopping training for now and seeking appropriate medical care or clinician guidance.

## Specificity over variety

Don't chase novelty. Repeating the same big lifts week-over-week with progressive overload beats a different exercise every session. Variety belongs in accessories and conditioning, not in the main lifts.

## Time budgeting

Rough split for a typical session:

- ≤ 30 min → 1 main block + brief warm-up. No finisher.
- 30–45 min → warm-up (5) + main (25–35) + finisher OR cooldown (5).
- 45–75 min → warm-up (8) + main (35–55) + finisher (5–10) + cooldown (5).
- 75+ min → can combine modalities (lift + zone 2) but only if it serves the goals.

## Heart rate prescriptions

Always give actual BPM ranges using the user's stored max HR. "Zone 2" means nothing without numbers. Example output: "Zone 2: 110–128 bpm" not "Zone 2 effort."

If the user has no max HR data, use 220 - age and note the assumption in the workout.

## Loading

For strength work, prescribe specific loads when possible — pull from the most recent log entry for that lift and progress sensibly:

- If last session was completed with reps in reserve → +5 lb (upper) or +10 lb (lower).
- If last session was a grinder → repeat the load.
- If last session failed reps → repeat or back off 5–10%.

If you have no log data for that lift, prescribe by RPE ("work up to a 5-rep RPE 7") and ask the user to log the actual load.

## Safety defaults

- Warm-up should ramp the modality of the main work (don't do static stretching before max-effort lifting; do dynamic warm-up + lighter ramp sets).
- Don't program max-effort sprinting or jumps cold.
- For barbell work, suggest a working-up scheme (e.g., 135 × 5, 185 × 3, 225 × 1, then working sets).
- Honor the injury list. If the user listed "left knee, no deep squatting" — no deep squats. Ever. Until they update the profile.
- Do not diagnose medical issues, prescribe rehab for unresolved injuries, or override clinician restrictions.

## Output discipline

Every workout must be runnable as-written. The user should not have to ask "how heavy?" or "how long?" or "what HR?" — those numbers go in the plan.
